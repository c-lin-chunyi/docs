# 课业材料目录

本部分主要记录浙江大学心理学培养方案上的必修、专业基础、专业必修，以及专业选修部分本人选修过的课程的课程大纲、[ECTS 分数映射](https://education.ec.europa.eu/education-levels/higher-education/inclusive-and-connected-higher-education/european-credit-transfer-and-accumulation-system)、评分标准、就读体验以及其他补充材料。

 部分材料由于学校的版权限制，无法直接提供下载。我设置了 Cloudflare Access 验证。在下载相关资料时，您可能会被提示需要使用邮箱验证。请输入您`@zju.edu.cn`后缀的邮箱，并输入接收到的验证码，方可通过验证并下载。验证有效时间为 30 分钟，期间不需要重复验证。
